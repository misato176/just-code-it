※1スクレイピング<br>
　WEBサイトから自動的に情報を取得する方法<br>
　方　法：プログラムもしくはスクリプトを使って<br>
　　　　　WEBページのHTMLデータを解析し、特定の情報を抽出する。<br>
　目　的：データ収集、競合分析、自動化、データ分析<br>
　注意点：利用規約を守ること　（robots.txtを確認する）<br>
  　　　　　負荷をかけない　（time.sleepで大量のリクエストを避ける）<br>
　　　　　法的リスク　著作権やプライバシーの問題に注意<br>
